## Overview of Projects

### [DO1_Linux](DO1_Linux) — Системное администрирование
Базовая настройка и администрирование серверов на базе **Ubuntu Server 20.04 LTS**.
- **Установка и настройка ОС:** Конфигурация LVM, создание разделов диска.
- **Управление пользователями:** Создание пользователей, групп, настройка прав доступа и `sudo`.
- **Сетевая конфигурация:** Настройка статических IP, шлюзов и DNS через `netplan`.
- **Безопасность:** Настройка **SSHD** (смена порта, ключи), анализ логов безопасности (`auth.log`).
- **Мониторинг и обслуживание:** Работа с процессами (`top`, `htop`), анализ дискового пространства (`df`, `du`, `ncdu`), настройка автоматических задач через **CRON**.

[Подробнее](DO1_Linux/README_DO1.md)

### [DO2_LinuxNetwork](DO2_LinuxNetwork) — Сетевая инженерия
Построение сложной сетевой инфраструктуры на виртуальных машинах, имитирующей реальные производственные среды.
**Решенные задачи:**
- **Адресация:** Работа с подсетями, расчет масок и диапазонов с помощью `ipcalc`.
- **Маршрутизация:** Настройка статической маршрутизации между несколькими сегментами сети (5 виртуальных машин, включая роутеры).
- **Сетевые сервисы:** Поднятие и конфигурация **DHCP-сервера** (`isc-dhcp-server`) с привязкой по MAC-адресам.
- **Безопасность сети:** Настройка межсетевого экрана **iptables** (правила INPUT/OUTPUT/FORWARD, стратегии DROP/ACCEPT).
- **NAT и проксирование:** Настройка **SNAT** и **DNAT**, проброс портов.
- **Туннелирование:** Организация **SSH-туннелей** (Local и Remote port forwarding) для доступа к закрытым ресурсам.

[Подробнее](DO2_LinuxNetwork/README_DO2.md)

### [DO3_LinuxMonitoring_v1](DO3_LinuxMonitoring_v1) — Автоматизация мониторинга
Разработка скриптов на **Bash** для сбора и анализа системных метрик.
**Решенные задачи:**
- **Сбор метрик:** Написание скриптов для получения данных о системе (CPU, RAM, HDD, Network) и форматированного вывода.
- **Анализ файловой системы:** Автоматический поиск самых больших файлов и директорий, классификация файлов по типам (conf, log, exe), вычисление хешей (MD5).
- **Работа с конфигурациями:** Реализация парсинга внешних конфигурационных файлов для управления цветовой схемой вывода терминала.
- **Оптимизация:** Замер времени выполнения скриптов и оптимизация алгоритмов поиска.

[Подробнее](DO3_LinuxMonitoring_v1/README_DO3.md)

### [DO5_SimpleDocker](DO5_SimpleDocker) — Контейнеризация
Создание микросервисной архитектуры с использованием **Docker**.
**Решенные задачи:**
- **Создание образов:** Написание оптимизированных **Dockerfile** (Multi-stage builds) для уменьшения размера итогового образа.
- **Веб-сервер:** Настройка **Nginx** внутри контейнера, проксирование запросов, кастомные конфиги.
- **FastCGI:** Разработка и деплой мини-сервера на **C** с использованием `spawn-fcgi`.
- **Оркестрация:** Описание инфраструктуры в **docker-compose.yml** (сети, volume-маппинг, зависимости сервисов).
- **Безопасность:** Аудит образов с помощью **Dockle** (проверка на CIS benchmarks, права пользователей, пустые пароли).

[Подробнее](DO5_SimpleDocker/README_DO5.md)

### [DO6_CICD](DO6_CICD) — CI/CD Pipeline
Настройка полного цикла автоматической сборки, тестирования и деплоя приложения.
**Решенные задачи:**
- **GitLab CI:** Настройка раннеров и написание пайплайна `.gitlab-ci.yml`.
- **Этапы пайплайна:**
    - **Build:** Автоматическая сборка проекта через `make`.
    - **Code Style:** Проверка стиля кода с помощью `clang-format`.
    - **Test:** Запуск интеграционных тестов.
    - **Deploy:** Автоматическая доставка артефактов на удаленный сервер через `scp` и `ssh`.
- **Уведомления:** Интеграция с **Telegram Bot API** для отправки отчетов о статусе сборки (успех/провал).
